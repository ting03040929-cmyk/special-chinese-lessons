# 🎯 國語文特教教材：互動式學習網頁製作指南

本指南說明如何將 Gemini 產出的特教版 Word 教材 (`gemini_[課文名稱].docx`) 與語詞配圖，快速轉換為**無須伺服器、支援離線雙擊即用**的「特教互動學習網頁」。

---

## 📂 1. 標準目錄結構

在每一課的資料夾下（例如 `E:\antigravity\202606_國文教材\錢是省出來的`），建立一個名為 `web/` 的子目錄：

```text
[課文名稱]/
├── 圖片/                          # 存放語詞配圖 (12個常用詞 + 8個難詞)
│   ├── [語詞A].png
│   └── ...
├── gemini_[課文名稱].docx         # 特教精簡版 Word 課文與練習題
└── web/                          # 互動網頁主目錄 (本指南製作的內容)
    ├── index.html                # 網頁結構與版面
    ├── style.css                 # 視覺設計與動畫樣式
    ├── app.js                    # 互動控制與語音/遊戲邏輯
    ├── data.js                   # 由 Python 自動解析產出的課文與題目數據
    └── images/                   # 從「圖片/」複製過來的 PNG 檔 (相對路徑讀取)
```

---

## ⚡ 2. 製作流程步驟

### 步驟 1：建立網頁資料夾與複製圖片
1. 在課文目錄下，新建 `web` 及 `web/images` 資料夾。
2. 將 `圖片/` 資料夾下的所有語詞配圖，**完整複製**到 `web/images/` 之下。

### 步驟 2：解析 Word 並生成 `data.js`
使用以下 Python 腳本，自動讀取 Word 檔，並解析出結構化的數據並寫入 `web/data.js`。

#### 🐍 解析腳本 `parse_docx.py`
請在課文目錄下執行此 Python 腳本（需安裝 `python-docx`，可透過 `uv run --with python-docx parse_docx.py` 執行）：

```python
import sys
import os
import docx
import json
import re

# 請修改為當前課文名稱與檔案名稱
LESSON_NAME = "錢是省出來的" 
DOCX_PATH = f"gemini_{LESSON_NAME}.docx"
OUTPUT_JS_PATH = os.path.join("web", "data.js")

def parse_docx():
    if not os.path.exists(DOCX_PATH):
        print(f"找不到 Word 檔案: {DOCX_PATH}")
        return
        
    doc = docx.Document(DOCX_PATH)
    paragraphs = [p.text.strip() for p in doc.paragraphs if p.text.strip()]
    
    story = []
    # 請依該課 Word 定義常用語詞
    common_vocabs = ["口袋", "柔軟", "衛生紙", "節儉", "毛巾", "肥皂", "漂亮", "衣服", "失望", "修理", "長大", "浪費"]
    difficult_vocabs = []
    fill_in_the_blanks = []
    matching_questions = []
    mcqs_blank = []
    mcqs_concept = []
    reading_comprehension = []
    
    # 1. 解析課文段落
    content_idx = -1
    for idx, text in enumerate(paragraphs):
        if "課文內容" in text:
            content_idx = idx
            break
            
    if content_idx != -1:
        idx = content_idx + 1
        while idx < len(paragraphs) and "一、 課文語詞挑選與注釋" not in paragraphs[idx]:
            story.append(paragraphs[idx])
            idx += 1

    # 2. 解析偏難語詞
    diff_idx = -1
    for idx, text in enumerate(paragraphs):
        if "8個偏難語詞注釋與例句" in text:
            diff_idx = idx
            break
            
    if diff_idx != -1:
        idx = diff_idx + 1
        while idx < len(paragraphs) and "二、 特教格式語詞學習單" not in paragraphs[idx]:
            line = paragraphs[idx]
            match = re.match(r'^([^\x00-\xff（]+)（([^）]+)）：(.*)$', line)
            if match:
                word = match.group(1).strip()
                zhuyin = match.group(2).strip()
                definition = match.group(3).strip()
                example = ""
                if idx + 1 < len(paragraphs) and paragraphs[idx + 1].startswith("課文例句："):
                    example = paragraphs[idx + 1].replace("課文例句：", "").strip()
                    idx += 1
                difficult_vocabs.append({
                    "word": word, "zhuyin": zhuyin, "definition": definition, "example_sentence": example
                })
            else:
                match2 = re.match(r'^([^\x00-\xff：]+)：(.*)$', line)
                if match2:
                    word = match2.group(1).strip()
                    definition = match2.group(2).strip()
                    zhuyin = ""
                    if word == "不合時宜": # 針對無注釋難詞的手動修正
                        zhuyin = "ㄅㄨˋ ㄏㄜˊ ㄕˊ ㄧˊ"
                    example = ""
                    if idx + 1 < len(paragraphs) and paragraphs[idx + 1].startswith("課文例句："):
                        example = paragraphs[idx + 1].replace("課文例句：", "").strip()
                        idx += 1
                    difficult_vocabs.append({
                        "word": word, "zhuyin": zhuyin, "definition": definition, "example_sentence": example
                    })
            idx += 1

    # 3. 解析填充題
    fill_start = -1
    for idx, text in enumerate(paragraphs):
        if "◎ 生字語詞-句子（填充題）" in text:
            fill_start = idx
            break
            
    if fill_start != -1:
        idx = fill_start + 1
        if "【語詞銀行】" in paragraphs[idx]:
            idx += 1
        while idx < len(paragraphs) and "◎ 生字語詞-句子概念（配對）" not in paragraphs[idx]:
            para_text = paragraphs[idx]
            parts = para_text.split('\n')
            if len(parts) >= 2:
                fill_in_the_blanks.append({"question": parts[0].strip(), "hint": parts[1].strip()})
            else:
                fill_in_the_blanks.append({"question": para_text.strip(), "hint": ""})
            idx += 1

    # 4. 解析連連看/配對題
    match_start = -1
    for idx, text in enumerate(paragraphs):
        if "◎ 生字語詞-句子概念（配對）" in text:
            match_start = idx
            break
            
    if match_start != -1:
        idx = match_start + 1
        while idx < len(paragraphs) and not paragraphs[idx].startswith("( 　 )") and "◎ 練習題" not in paragraphs[idx]:
            idx += 1
        while idx < len(paragraphs) and "◎ 練習題" not in paragraphs[idx]:
            para_text = paragraphs[idx]
            if para_text.startswith("( 　 )"):
                parts = para_text.split('\n')
                q_text = parts[0].replace("( 　 )", "").strip()
                hint_text = parts[1].strip() if len(parts) >= 2 else ""
                matching_questions.append({"question": q_text, "hint": hint_text})
            idx += 1

    # 5. 解析挖空選擇題
    mcq_blank_start = -1
    for idx, text in enumerate(paragraphs):
        if "第一大題：生字語詞-句子（挖空選擇題）" in text:
            mcq_blank_start = idx
            break
            
    if mcq_blank_start != -1:
        idx = mcq_blank_start + 1
        while idx < len(paragraphs) and "第二大題：生字語詞-句子概念（選擇題）" not in paragraphs[idx]:
            para_text = paragraphs[idx]
            if para_text.startswith("( 　 )"):
                parts = para_text.split('\n')
                q_text = parts[0].replace("( 　 )", "").strip()
                hint_text = ""
                options = []
                for part in parts[1:]:
                    part = part.strip()
                    if part.startswith("【提示："):
                        hint_text = part
                    elif "(A)" in part:
                        opts = re.findall(r'\([A-D]\)\s*([^\(]+)', part)
                        options = [o.strip() for o in opts]
                mcqs_blank.append({"question": q_text, "hint": hint_text, "options": options})
            idx += 1

    # 6. 解析概念選擇題
    mcq_concept_start = -1
    for idx, text in enumerate(paragraphs):
        if "第二大題：生字語詞-句子概念（選擇題）" in text:
            mcq_concept_start = idx
            break
            
    if mcq_concept_start != -1:
        idx = mcq_concept_start + 1
        while idx < len(paragraphs) and "三、 閱讀理解測驗選擇題" not in paragraphs[idx]:
            para_text = paragraphs[idx]
            if para_text.startswith("( 　 )"):
                parts = para_text.split('\n')
                q_text = parts[0].replace("( 　 )", "").strip()
                hint_text = ""
                options = []
                for part in parts[1:]:
                    part = part.strip()
                    if part.startswith("【提示："):
                        hint_text = part
                    elif "(A)" in part:
                        opts = re.findall(r'\([A-D]\)\s*([^\(]+)', part)
                        options = [o.strip() for o in opts]
                mcqs_concept.append({"question": q_text, "hint": hint_text, "options": options})
            idx += 1

    # 7. 解析閱讀測驗
    rc_start = -1
    for idx, text in enumerate(paragraphs):
        if "三、 閱讀理解測驗選擇題" in text:
            rc_start = idx
            break
            
    if rc_start != -1:
        idx = rc_start + 1
        current_section = ""
        while idx < len(paragraphs):
            para_text = paragraphs[idx]
            if para_text.startswith("【") and para_text.endswith("】"):
                current_section = para_text
            elif para_text.startswith("( 　 )"):
                parts = para_text.split('\n')
                q_text = parts[0].replace("( 　 )", "").strip()
                options = []
                for part in parts[1:]:
                    part = part.strip()
                    if "(A)" in part:
                        opts = re.findall(r'\([A-D]\)\s*([^\(]+)', part)
                        options = [o.strip() for o in opts]
                reading_comprehension.append({
                    "section": current_section, "question": q_text, "options": options
                })
            idx += 1

    data = {
        "lesson_name": LESSON_NAME,
        "story": story,
        "common_vocabs": common_vocabs,
        "difficult_vocabs": difficult_vocabs,
        "fill_in_the_blanks": fill_in_the_blanks,
        "matching_questions": matching_questions,
        "mcqs_blank": mcqs_blank,
        "mcqs_concept": mcqs_concept,
        "reading_comprehension": reading_comprehension
    }
    
    js_content = f"const lessonData = {json.dumps(data, ensure_ascii=False, indent=2)};\n"
    
    os.makedirs(os.path.dirname(OUTPUT_JS_PATH), exist_ok=True)
    with open(OUTPUT_JS_PATH, 'w', encoding='utf-8') as f:
        f.write(js_content)
        
    print(f"成功生成 {OUTPUT_JS_PATH}！")

if __name__ == '__main__':
    parse_docx()
```

### 步驟 3：部署靜態網頁核心檔案
將預先準備好的 `index.html`、`style.css` 及 `app.js` 範本檔案，複製並放入新課文的 `web/` 資料夾內。

## ⚙️ 3. 核心修改指南 (適用於新課文)

若新課文的題目數量或常用語詞不同，通常只需在 `app.js` 及 `data.js` 中調整以下兩點：

### A. 段落與語詞對應 (`app.js`)
在 `app.js` 的 `paragraphVocabs` 物件中，手動定義**哪一個段落（從 0 開始）對應到哪些語詞**，控制當學生點選該段落時，右側側邊欄要顯示哪些語詞圖片：

```javascript
// 修改此對應以符合新課文
const paragraphVocabs = {
  0: ["口袋", "柔軟"],
  1: ["衛生紙", "節儉", "毛巾", "山窮水盡"],
  2: ["肥皂", "高雅", "漂亮", "土裡土氣"],
  3: ["衣服", "克己", "將就"],
  4: ["修理", "開口笑", "自卑"],
  5: ["長大", "浪費"],
  6: ["不合時宜", "失望"]
};
```

### B. 常用語詞的白話字典定義 (`app.js`)
常用語詞在 Word 中通常沒有附注釋與例句。為了讓網頁的點擊彈窗功能有資料可以顯示，需在 `app.js` 的 `commonVocabDetails` 中定義這些語詞的白話解釋：

```javascript
const commonVocabDetails = {
  "語詞A": { definition: "白話文的定義", example: "簡單生活化的課文例句" },
  ...
};
```

### C. 答案庫設定 (`app.js`)
在 `app.js` 的 `answerKeys` 中，填入新課文的題目答案（以文字字串比對，以防題目選項順序隨機變動）：

```javascript
const answerKeys = {
  fillblank: ["填充題第1題答案", "第2題答案", ...],
  concept: ["選擇題第1題答案", ...],
  blank: ["挖空題第1題答案", ...],
  comprehension: ["閱讀測驗第1題答案", "第2題答案", ...]
};
```

## 💻 4. 教學與發布方式

製作完成後，有兩種方式可以使用該網頁：

1.  **無網離線包**：
    *   將整個 `web/` 資料夾壓縮為 `.zip` 檔發送給學生或匯入平板。
    *   解壓縮後，直接在瀏覽器雙擊 `index.html` 即可開啟，完全不需要連接網際網路。
2.  **架設本地伺服器**：
    *   若想在教室內以網址分享，可以在 `web/` 資料夾下開啟終端機並執行 Python 的簡易伺服器：
        ```powershell
        python -m http.server 8000
        ```
    *   學生即可透過 `http://[教師電腦IP]:8000` 連入遊玩。
